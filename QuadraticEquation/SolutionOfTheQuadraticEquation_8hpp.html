Функция для решения квадратных уравнений вида a*x^2 + b*x + c = 0.

SolveSquare(double a, double b, double c, double *x1, double *x2, size_t *n_roots)

Аргументы
[in]a- коэффициент при x^2
[in]b- коэффициент при x
[in]c- свободный член
[out]x1- указатель на первый корень
[out]x2- указатель на второй корень
[out]n_roots- указатель на количество корней
Заметки
Если n_roots равен 3, то это значит, что уравнение имеет бесконечно много корней

Уравнение c = 0 при c <> 0 не имеет решения
